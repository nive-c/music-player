# 🎵 Music Player

A clean, aesthetic music player web app built with HTML, CSS, and JavaScript.

## ✨ Features
- Play, pause, skip previous and next
- Progress bar with click-to-seek
- Volume control slider
- Playlist with active song highlight
- Auto-plays next song when current ends
- Album cover display per song

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (Web Audio API)

## 🚀 How to Run
1. Clone or download the repository
2. Add your `.mp3` files to the project folder
   (filenames should match the `src` fields in the songs array)
3. Open `index.html` in your browser

> ⚠️ Audio files are not included in this repo due to copyright.
> Replace the `src` values in the songs array with your own mp3 files.

## 💼 What This Shows
- Working with the HTML Audio API
- Dynamic DOM manipulation
- Event-driven JavaScript
- Clean UI design with CSS variables
